# User-Based-Movie-Recommendation-System---ML

This project is a Machine Learning based recommendation system that suggests movies by finding users with similar preferences and recommending movies they liked.

Features

💠 Recommend movies based on similar users’ preferences
💠 Uses user-item similarity (collaborative filtering)
💠 Personalized movie recommendations
💠 Built with Machine Learning and Streamlit

Technologies

Python • Pandas • NumPy • Scikit-learn • Streamlit

Use Case

Helps provide personalized movie recommendations based on user behavior and ratings.
